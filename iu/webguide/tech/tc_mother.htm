MOTHERSOFT

"MotherSoft is a loose-knit organization devoted to making good, solid,useful software freely available to the general public. We despise counter-intuitive hidden source code, trade secrets, snobby copyrights and other elements of tight-wad software development that has made a dark blotch in the history of computer software." These friendly programmers give code away free. You'll find graphical finger PERL scripts, a free CGI utility that generates multiple inline GIF images from one image source file, a random URL generator and much more.
