UNIXHELP FOR USERS

UNIX can be daunting at the outset. The commands are not even close to English, and even if you know how to use the online manuals, you often don't know what to start looking under. This site explains many aspects of UNIX in simple language. You will learn about shell scripts, e-mail, managing processes, commands and file transfers. An immensely useful resource for anyone with an unfathomable shell account.
