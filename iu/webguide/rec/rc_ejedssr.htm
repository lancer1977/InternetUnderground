EVIL JED'S SURF REPORT

OK, so this doesn't help a lot if you're not in San Diego, but Evil Jed has nonetheless taken a lot of HTML toys and created a site that actually might be of some use to surfers. And we're not talking about the Web kind--we're talking real surfers, with boards and tans and an individually developed lexicon that doesn't include emoticons. With a real-time picture of Scripps Pier and predicted swell heights for spots around San Diego, dedicated surfers can use the Web to predict whether today's surf conditions warrant a trip to the beach or not.
