Buyers Tip Sheet!
What you really need to get on the 'net, and pitfalls to watch for when buying a used computer...

WHAT WHAT YOU SHOULD LOOK FOR POSSIBLE PITFALLS EXPECT TO PAY
SCREEN Non-interlaced monitor with dot-pitch of at least .28mm; color, even if it's more 14" too small for serious work $100-300
PROCESSOR For PCs, look for a 386 or faster; 25MHz will run Netscape or online software, though slowly Dead-end upgrade path $500-1000
HARD DRIVE Minimum of 80MB, try for 120 Tempting to use DriveSpace, DON'T!! $75-100
RAM Minimum of 4MB RAM to run Windows-based software and deal with graphics (8MB for Windows 95) Older boards use 30 pin RAM which is obsolete $30-40 per MB
BRAND NAME Compaq Aero computer Keyboard takes some getting used to; Required external floppy drive $650-750
MODEM Try for 14.4, which should not be too expensive since people are switching over to 28.8; Preferably U.S. Robotics, Hayes Watch out for serial port problems with 9600+ modems < $100
