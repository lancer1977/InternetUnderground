Drowned God Patch v.5

Once the .EXE file is on your hard drive, double-click
it's icon to automatically extract the contents of the 
patch. Run "Setup.exe" and follow the on-screen
instructions to install the patch.

This patch corrects all the problems fixed by previous
patches and fixes a rare problem with a screen freeze 
at the beginning of the game with the valve-operated
computer. If you have already installed Patch 4 and 
are not experiencing this problem, you don't need this
patch.
